Programação em C

Programação em C é uma das linguagens mais populares e amplamente utilizadas no desenvolvimento de software, com estruturas de controle, funções, arrays, ponteiros e manipulação de arquivos.

Nesta página vou ensinar o básico sobre a programação em C.

1. Bibliotecas, Função main e Tipos de Dados

Em C, as bibliotecas são conjuntos de funções pré-definidas que facilitam a programação. A função main é o ponto de entrada de um programa em C. Os tipos de dados incluem int, float, char, entre outros.

Exemplo de uso:


#include <stdio.h>

int main() {
    printf("Ola Mundo!");
    return 0;
}

2. Entrada/Saída e Variáveis

A entrada e saída de dados em C são realizadas principalmente pelas funções scanf e printf.

As variáveis devem ser declaradas com um tipo específico antes de serem usadas.

Exemplo de uso:


int x;
scanf("%d", &x);
printf("%d", x);

Comando de Atribuição

O comando de atribuição usa o símbolo (=). Exemplo: x = 10;

Exemplo de uso:


int x;
x = 10;

Comando de Condição

Os comandos if, else, else if e switch permitem tomar decisões no programa.

Exemplo de uso:


if (x > 10) {
    printf("Maior que 10");
}

Strings

Em C, uma string é um vetor de caracteres char terminado com '\0'.

Exemplo de uso:


char nome[20] = "Raphael";

Estruturas de Repetição

As estruturas for, while e do-while permitem repetir blocos de código.

Exemplo de uso:


for(int i = 0; i < 5; i++) {
    printf("%d", i);
}
